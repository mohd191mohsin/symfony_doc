Project Setup

.env
  1. update database configuration in .env file
    • DATABASE_URL="mysql://root:@127.0.0.1:3306/symfony_crud"
    • Run the following command to create the database: php bin/console doctrine:database:create
      • What is Doctrine

        Doctrine is an Object-Relational Mapper (ORM) for PHP that provides a powerful and flexible way to interact with databases using object-oriented programming principles. It acts as a bridge between the object-oriented domain model of your application and the relational database, allowing developers to work with database records as PHP objects rather than writing raw SQL queries.

  2. Create Model or Entity
    • Run the following command to create an entity class: php bin/console make:entity
    • Follow the prompts to define the entity's fields and their types. 
          Add another property? Enter the property name (or press  to stop adding fields):
    > name
    Field type (enter ? to see all types) [string]:
    > string
    Field length [255]:
    > 255

    > no
    -----
    Add another property? Enter the property name:
    > price
    Field type:
    > float
    Nullable? [no]:
    > no
    -----
    Add another property? Enter the property name:

    Field type:
    > text
    Nullable? [no]:

  3. After defining the entity, run the following command to create a migration file that will generate the corresponding database table: php bin/console make:migration
  4. Run the migration to create the table in the database: php bin/console doctrine:migrations:migrate
