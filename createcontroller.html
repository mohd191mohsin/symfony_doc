Create Controller By Command

  • Open Terminal And Run command
    • php bin/console make:controller HomeController
      • This command create a file in src/Controller/HomeController.php
      • It create a HomeController final class with extends AbstractController
      • It create a index method with return render Template and create a named route e.g "/home" and name "app_home"
      • It create a twig file in templates/home/index.html.twig that extends the base.html.twig file
      • In Base file it create a blocks like title and body
Controller Code
                    
                    <?php
                        namespace App\Controller;

                        use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
                        use Symfony\Component\HttpFoundation\Response;
                        use Symfony\Component\Routing\Attribute\Route;

                        final class HomeController extends AbstractController
                        {
                            #[Route('/home', name: 'app_home')]
                            public function index(): Response
                            {
                                return $this->render('home/index.html.twig', [
                                    'controller_name' => 'HomeController',
                                ]);
                            }
                        }
