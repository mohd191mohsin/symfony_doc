Create Controller Manually

  • Creat a file in a controller folder e.g TestController.php
    • Add namespace App\Controller;
    • Import use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
    • Import use Symfony\Component\HttpFoundation\Response;
    • Import use Symfony\Component\Routing\Attribute\Route;
    • create a class e.g TestController with extends AbstractController
    • create a method e.g index with return type Response
    • use Route attribute to define a route e.g "/test" and name "app_test"
    • use Response to show data
    • create a twig file in templates/test/index.html.twig that extends the base.html.twig file
    • Test on http://127.0.0.1:8000/test
Controller Code
                    
                    <?php
                        
                        namespace App\Controller;

                        use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
                        use Symfony\Component\Routing\Attribute\Route;
                        use Symfony\Component\HttpFoundation\Response;

                        final class TestController extends AbstractController{
                            #[Route('test',name:'app_test')]
                            public function index(){
                                return new Response('<h1>testing</h1>');
                            }
                        }
