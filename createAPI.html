Create API

  • Create and Run Project
    1. symfony new API
    2. cd API
    3. Install Required Packages using composer composer require orm doctrine maker serializer
    4. symfony server:start
    5. Goto http://127.0.0.1:8000/
    6. open same terminal
    7. open folder in vs codecode .
  • Connect With DataBase
    1. Open .env file
    2. Change DATABASE_URL="mysql://root@127.0.0.1:3306/symfony_API"
  • Create DataBase
    1. php bin/console doctrine:database:create
  • Create Model OR Entity For Product
    1. php bin/console make:entity Product
    2. Add following fields
      • name:string
      • price:float
      • created_at:datetime_immutable
      • updated_at:datetime
    3. php bin/console make:migration
    4. php bin/console doctrine:migrations:migrate
